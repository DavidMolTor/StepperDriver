# STEPPER DRIVER

A simple stepper driver module using a Seeed Studio Xiao development board and TMC2208 stepper driver IC.

## Introduction

This module is designed to be used as a stepper controller for a testbench platform.

The design is centered around the ESP32-C6 XIAO development board from Seeed Studio. It can be soldered as an SMD part in order to have access to the driver debug wire and JTAG connector on the board.

Compatibility with other drivers like the TMC2209 could be possible, the footprints for the different package needs to be checked before attempting to use it.

## Features

 - Wide input voltage range of 12-36V.

 - High current capability of up to 1.2A.

 - Wireless connectivity and LCD screen.

 - Afforable BOM with high quality parts.
